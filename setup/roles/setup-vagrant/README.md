vagrant
=======

Test if the /usr/local/bin/vagrant exists and fail the role if it doesn't
with a message to install vagrant

Requirements
------------

Vagrant needs to be installed for this role to succeed.

Role Variables
--------------

None

Dependencies
------------

None

Example Playbook
----------------

    - hosts: localhost
      roles:
         - role: vagrant
